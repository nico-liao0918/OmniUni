# Omniuni

**A single repository containing all my web projects, structured as modular sub-sites with a unified design system.**

## Overview

Omniuni is the centralized repository for all my static websites and web projects. It provides:

- Modular sub-sites: Each project is maintained in its own folder for clarity and scalability.
- Unified design system: Shared CSS, JS, and assets ensure consistency across all sites.
- Easy management: Centralized updates and reusable components save development time.

This repository is perfect for organizing multiple websites in one place without losing modularity or flexibility.

## Features

- Centralized management of multiple web projects
- Shared design system for consistent look and feel
- Modular sub-sites for independent development and deployment
- Easily extensible — new sites can be added without disrupting the ecosystem

## Future Plans

- Integrate automated build system for all sub-sites
- Implement shared components library for rapid prototyping
- Set up deployment scripts to streamline publishing
